# PageMarker - 页码标注工具

基于 .NET Framework 4.7.2 的图片页码批量标注工具。

## 功能概述

将图片添加页码文字，支持描边、缩放、左右分页等多种配置，保持原始图片高清画质输出。

## 目录结构

```
程序根目录/
├── imgsrc/          ← 放入原始图片（支持 jpg/png/bmp）
├── output/          ← 输出带页码的图片（自动创建）
└── config.ini       ← 配置文件（自动生成）
```

## 使用方法

1. 在程序目录下创建 `imgsrc` 文件夹，放入需要添加页码的图片
2. 图片按文件名数字顺序排列（如 `0.jpg, 1.jpg, 2.jpg ... 10.jpg, 11.jpg`）
3. 调整右侧参数配置
4. 点击「生成预览图」预览第一张效果
5. 点击「开始批量处理」批量输出到 `output` 文件夹

## 配置项说明

### 风格设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 选择字体 | 页码文字字体 | 微软雅黑 |
| 字体大小 | 文字大小（像素） | 36 |
| 描边大小 | 文字描边宽度（像素） | 3 |
| 颜色选择 - 字色 | 文字填充颜色 | 白色 |
| 颜色选择 - 描边 | 文字描边颜色 | 黑色 |
| 加粗 | 文字是否加粗 | 否 |

### 距离边缘

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 横向边距 | 页码距左右边缘的距离（px） | 30 |
| 纵向边距 | 页码距底部边缘的距离（px） | 30 |
| 缩放边距 | 图片缩小比例（%） | 0 |

### 页设置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 图片起始页数 | 第一张加页码图片的页码编号 | 1 |
| 文件起始页数 | 从第几张图片开始加页码（之前的原样复制） | 1 |
| 左右分页(左开始) | 奇数页左下角，偶数页右下角 | ✅ |
| 左右分页(右开始) | 奇数页右下角，偶数页左下角 | |
| 居右下角 | 所有页码统一在右下角 | |
| 不加页码的图片不缩放 | 跳过的页面是否应用缩放 | 否 |

## 页码位置模式

### 左右分页(左开始)
- 第 1 页 → 左下角
- 第 2 页 → 右下角
- 第 3 页 → 左下角
- ...交替

### 左右分页(右开始)
- 第 1 页 → 右下角
- 第 2 页 → 左下角
- 第 3 页 → 右下角
- ...交替

### 居右下角
- 所有页码统一在右下角

## 缩放边距

输入百分比值（0~100），将原图缩小后居中放置，四周用白色填充：

| 输入值 | 效果 |
|--------|------|
| 0 | 不缩放，原图处理 |
| 5 | 原图缩小到 95%，居中白色填充 |
| 10 | 原图缩小到 90%，居中白色填充 |

配合「不加页码的图片不缩放」选项，可以控制跳过页是否也应用缩放。

## 跳过页说明

当「文件起始页数」大于 0 时，前面的图片为跳过页（如封面），不加页码。

例如 `imgsrc` 中有 `0.jpg, 1.jpg, 2.jpg, 3.jpg`，文件起始页数设为 1：

| 输入文件 | 输出文件 | 处理方式 |
|---------|---------|---------|
| 0.jpg | 0.jpg | 跳过页（不加页码） |
| 1.jpg | 1.jpg | 加页码 1 |
| 2.jpg | 2.jpg | 加页码 2 |
| 3.jpg | 3.jpg | 加页码 3 |

## 输出格式

- PNG 图片：无损保存
- JPG 图片：100% 质量编码
- 输出文件名按序号命名，与输入文件扩展名一致

## 配置

所有配置项在程序关闭时自动保存到 `config.ini`，下次启动自动恢复。
